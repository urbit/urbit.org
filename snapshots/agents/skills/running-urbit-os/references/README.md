# Running Urbit OS references

This directory holds optional reference documents for the `running-urbit-os` core skill.

## How to contribute

1. Add one focused markdown file here.
2. Prefer filenames like `<author>-<topic>.md`.
3. Link upstream docs, repos, or specs where helpful.
4. Add one bullet to `../SKILL.md` so the new reference is discoverable.

Do not add nested `SKILL.md` files under this directory.
