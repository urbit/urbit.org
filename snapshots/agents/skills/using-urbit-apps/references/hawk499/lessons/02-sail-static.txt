::  Lesson 02: raw handler + SAIL static page
::
::  Install path: /learn/02-sail-static
::  View URL:     /o/learn/02-sail-static
::
::  New idea:
::    A normal endpoint is a gate from `http-request` to a strand form.
::    SAIL (`;html`, `;body`, `;h1`) produces `manx` HTML trees.
::
|=  req=http-request
=/  m  (strand ,~)
^-  form:m
=*  r  ~(. server req ~)
%-  respond-html:r
;html
  ;head
    ;meta(charset "UTF-8");
    ;meta(name "viewport", content "width=device-width, initial-scale=1");
    ;link(rel "stylesheet", href "/hawk499/assets/feather");
    ;title: Lesson 02 — SAIL
  ==
  ;body.page.prose
    ;h1: Lesson 02 — SAIL static page
    ;p
      ; This page is a raw handler. It receives `req`, creates the `r`
      ; response helper door, and calls `respond-html:r` with a SAIL tree.
    ==
    ;ul
      ;li: `;html` creates an html node.
      ;li: `;body.page.prose` creates a body node with classes.
      ;li: `respond-html:r` serializes the `manx` tree and sends it to Eyre.
    ==
  ==
==
