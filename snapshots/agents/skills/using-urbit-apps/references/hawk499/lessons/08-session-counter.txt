::  Lesson 08: per-user session counter
::
::  Install path: /learn/08-session-counter
::  View URL:     /o/learn/08-session-counter
::
::  New idea:
::    `get-session:r` and `set-session:r` store per-user state for this endpoint.
::
|=  req=http-request
=/  m  (strand ,~)
^-  form:m
=*  r  ~(. server req ~)
|^
?:  is-post:r
  =/  form=(map @t @t)  formencoded-body:r
  =/  op=@t  (fall (~(get by form) 'op') 'inc')
  ;<  old=(unit @ud)  bind:m  (get-session:r @ud)
  =/  n=@ud  (fall old 0)
  =/  next=@ud  ?:  =('reset' op)  0  +(n)
  ;<  ~  bind:m  (set-session:r !>(next))
  (respond-redirect:r 303 (crip (spud base.req)))
?.  (is-method:r 'GET')
  (respond-json-error:r 405 'method not allowed')
;<  old=(unit @ud)  bind:m  (get-session:r @ud)
(respond-html:r (page (fall old 0)))
