::  Lesson 12: public versus owner-only endpoints
::
::  Install path: /learn/12-public-vs-owner-only
::  View URL:     /o/learn/12-public-vs-owner-only
::
::  New idea:
::    Runtime endpoints are public unless your handler checks `our` versus `src`.
::
::  Try:
::    Add `?owner-only` to the URL. If you are not authenticated as the host
::    ship, the endpoint returns 403.
::
|=  req=http-request
=/  m  (strand ,~)
^-  form:m
=*  r  ~(. server req ~)
=/  owner-only=?  (~(has by query.req) 'owner-only')
?.  |(!owner-only =(our src):req)
  (respond-payload:r [403 ~] ~)
